# InvestLens Repository Instructions

이 저장소에서 작업하는 코딩 에이전트는 다음 규칙을 준수합니다.

1. 작업 전 `CONTRIBUTING.md`를 읽고 아키텍처, 검증, 커밋 규칙을 따릅니다.
2. 안전하고 되돌릴 수 있는 로컬 작업은 승인 요청 없이 구현부터 검증까지 완료합니다.
3. 변경을 목적별로 분류하고 관련 테스트를 함께 커밋합니다.
4. 작업이 완료되면 별도 요청을 기다리지 않고 검증된 변경을 `<type> : <한국어 요약>` 형식으로 커밋합니다.
5. 전체 작업 종료 전 `./gradlew clean build`와 `git diff --check`를 실행합니다.
6. 검증을 통과한 커밋은 `origin`에 push합니다. 강제 push나 기록 재작성은 명시적인 요청 없이는 금지합니다.
7. 사용자 변경을 임의로 되돌리거나 삭제하지 않습니다.
8. 비밀키, `.env`, 운영 자격 증명은 절대 커밋하지 않습니다.
